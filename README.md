# API-FACULDADE

Atividade mensal (Projeto-2), uma API utilizando Node.js.

## Features

- [x] Cadastrar de Professor
- [x] Cadastrar de Turma
- [x] Cadastrar de Curso
- [x] Cadastrar de Aluno

- [x] Editar Professor
- [x] Editar Turma
- [x] Editar Curso
- [x] Editar Aluno

- [x] Remover Professor
- [x] Remover Turma
- [x] Remover Curso
- [x] Remover Aluno

- [x] Listar Professor
- [x] Listar Turma
- [x] Listar Curso
- [x] Listar Aluno
